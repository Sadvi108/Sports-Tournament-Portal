# Sports Tournament Portal (D-Clix Prototype)

Frontend-only prototype of a multi-sport tournament + club management portal with a marketing site and a `/competition` section.

## Features

- 18-sport selector + sport-filtered modules
- Watch Live (live + scheduled matches using mock data)
- New Club Registration (full form + notice content)
- Player Registration (tournament cards → registration flow)
- Marketing pages (Home / About / Pricing / Edu-Hub / Referral)
- Dark theme (Deep Black + Red + White) with rose-pink accents

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS
- Framer Motion (scroll reveal)
- tsParticles (Sparkles background)

## Run Locally

```bash
npm install
npm run dev
```

Vite prints the exact localhost URL (it may auto-increment ports if one is in use).
